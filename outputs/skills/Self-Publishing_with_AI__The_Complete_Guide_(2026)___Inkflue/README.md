## Vibe Coding 活用プロンプト

1. KDPで出版した電子書籍のランディングページを生成するHTML、CSS、JavaScriptコードを記述してください。本のタイトル、著者名、説明、表紙画像、購入リンクをカスタマイズ可能にしてください。
2. KDPの売上データを自動的に取得し、月ごとの収益をグラフ化するPythonスクリプトを作成してください。データはCSV形式で提供されると仮定します。
3. Markdown形式の原稿をEPUB形式に変換するためのシンプルなコマンドラインツールをGo言語で実装してください。目次生成と画像埋め込みに対応させてください。
