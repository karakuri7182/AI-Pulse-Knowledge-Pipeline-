## Vibe Coding 活用プロンプト

1. n8nのカスタムノードとして、特定のAPIからデータを取得し、整形するPythonスクリプトを生成してください。エラーハンドリングとログ出力を含めてください。
2. Makeシナリオで処理されたデータをGoogle Sheetsに書き込むためのJavaScript関数を作成してください。シート名と列名を引数として受け取り、既存の行を更新または新規追加するロジックを実装してください。
3. 社内システムとn8nを連携させるためのRESTful APIエンドポイントの仕様書（OpenAPI/Swagger形式）を生成してください。ユーザー認証、データ送信、ステータス更新の3つのエンドポイントを含めてください。
